A simple, secure document storage application built with Ruby on Rails 8. 

Users can create folders, upload files, and keep everything organized in one place. 

This MVP focuses on clarity, usability, and clean architecture while preparing for future enhancements.

## Core Functionality
1. User authentication (Devise)
2. Create and manage folders
3. Upload files into specific folders (Active Storage)
4. View files inside each folder
5. Clean, minimal UI inspired by Google Drive

## Tech Stack
1. Ruby on Rails 8
2. PostgreSQL
3. Devise for authentication
4. Active Storage for file uploads
5. Pico CSS + Bootstrap Utilities for styling
